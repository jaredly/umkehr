Let's support multi-select in our examples/block-rich-text app. cmd+click-drag should add a selection, or cmd+click to add a cursor. Triple click a word to select all occurrences of that word across all blocks. Typing, deleting, splitting should all work. Cursors that end up coinciding (visually, even if they are anchored to different logical nodes, e.g. in the presence of tombstones) should deduplicate with the "logical first" one winning.
