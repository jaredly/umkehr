WHen copying and pasting, we should recreate associated marks.
So copying bolded text, when you paste it, it should be bolded.
Copying text with an annotation mark, the annotation mark should be retained -- and point at the same annotation block.
However, if yuo've copied & pasted into a different document, the whole annotation will need to be re-created. So the "copied data" needs to include any annotation blocks, and pasting should create them if they don't exist. But if they do exist, it should not overwrite them.
