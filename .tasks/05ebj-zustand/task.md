examples/block-rich-text: let's try adopting zustand for state management. Currently perf takes a huge hit when there are lots of blocks on the page because we re-render every block on every selection change (recomputing a bunch of things besides). I'm hoping that zustand will allow us to do much more targeted updates, only re-rendering impacted blocks when text or selection changes.
